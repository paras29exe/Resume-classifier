## About the Project  
A web-based application that analyzes resumes to evaluate skill sets, predict job roles, and suggest improvements. It uses NLP-based parsing to extract essential information and generate analytics that help applicants understand how their resumes align with industry expectations.  

### Key Objectives  
- Convert resume data into structured tabular and CSV formats for analytics.  
- Provide actionable recommendations and predictions to improve resume quality.  
- Offer recruiters and colleges insights into applicants' strengths and interests.  
- Visualize analytics such as experience levels, skills, and location trends.

## Features  

### **Client**  
- Parse resumes to fetch:  
  - Basic Information  
  - Skills  
  - Keywords  
- Generate recommendations such as:  
  - Additional skills to learn  
  - Predicted job role  
  - Relevant courses and certificates  
  - Resume score and improvement tips  
  - Career guidance videos  

### **Admin**  
- Access all applicant data in tabular form.  
- Export applicant data as CSV.  
- Manage uploaded resumes.  
- View analytics dashboards including:  
  - Ratings  
  - Predicted fields/roles  
  - Experience levels  
  - Resume scores  
  - Geographic distribution  

### **Feedback Module**  
- Collect user feedback and ratings.  
- Display overall rating analytics and past comments.
